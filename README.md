### Animation format

### Interval
Interval is in ticks.
1 tick = 50 ms.

### Config
File:
config/animatedchat-client.toml

Main options:
- enable-animations ‚Äî enable animations
- history-size ‚Äî chat history size
- max-chat-length ‚Äî max input length
